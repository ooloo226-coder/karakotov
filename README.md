# Movie Library - Личная кинотека

**Автор:** [Ваше Имя Фамилия]

## Описание

Приложение для ведения личной коллекции фильмов с графическим интерфейсом. Позволяет добавлять, удалять, фильтровать и сохранять информацию о фильмах.

### Возможности

- Добавление фильмов с указанием названия, жанра, года выпуска и рейтинга
- Проверка корректности ввода (год - число, рейтинг 0-10)
- Табличное отображение всех фильмов
- Фильтрация по жанру (частичное совпадение) и по году (точное совпадение)
- Удаление выбранного фильма
- Автоматическое сохранение в JSON-файл
- Загрузка данных при запуске

## Установка и запуск

### Требования
- Python 3.6 или выше
- Стандартные библиотеки (tkinter, json, os) - входят в состав Python

### Запуск

1. Сохраните файл `movie_library.py`
2. Откройте терминал в папке с файлом
3. Выполните команду:
```bash
python movie_library.py
